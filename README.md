A Python-based utility for emergency departments and surgical wards to standardize Glasgow Coma Scale (GCS) assessments.

## Features
* **Interactive GCS Scoring:** Step-by-step input for Eye, Verbal, and Motor responses.
* **Automatic Severity Classification:** Categorizes injury as Mild, Moderate, or Severe based on the total score.
* **Clinical Export:** Generates a timestamped `.txt` report for patient documentation.
* **Neurological Logic:** Prevents invalid score combinations.
